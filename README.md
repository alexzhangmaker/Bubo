# Bubo

A new GitHub project.

## Components
- **Bubo Dashboard**: Open `console.html` to browse your trade history, reminders, and collected URLs.
- **BuboAgent**: The backend engine (Express/Mastra/Telegram).
- **Index**: The main entry point `index.html`.

## Getting Started
1. Run BuboAgent: `cd BuboAgent && npm start`
2. Open `console.html` in your browser.
