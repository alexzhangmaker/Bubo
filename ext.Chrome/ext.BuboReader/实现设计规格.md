# ext.BuboReader 实现设计规格

## 1. 项目概述
`ext.BuboReader` (BuboReader - Smart Reader & Note) 是一款功能强大的浏览器扩展，旨在为用户提供智能阅读模式、富文本笔记、云端同步以及 EPUB 导出功能。它结合了内容提取引擎、专门的阅读页面和侧边栏笔记管理，提供无缝的阅读与学习体验。

## 2. 核心架构
本扩展采用 Chrome Extension Manifest V3 架构，主要由以下组件构成：

### 2.1 Background Service Worker (`background.js`)
*   **职责**：全局状态管理、生命周期协调、API 集成。
*   **功能**：
    *   初始化站点选择器（Site Selectors）和自动启用模式（URL Patterns）。
    *   管理侧边栏（Side Panel）的显示行为。
    *   处理跨组件消息：打开阅读器标签页、获取/存储配置、处理 Google OAuth2 身份验证。
    *   打开专用阅读页面 `readability/reader.html` 并通过 URL 参数传递内容。

### 2.2 Content Script (`content.js`)
*   **职责**：网页内容提取与 UI 注入。
*   **功能**：
    *   **浮动工具栏**：在匹配 URL Patterns 的页面注入悬停按钮。
    *   **内容提取引擎**：
        1.  优先尝试用户定义的 CSS 选择器（Crane 模式）。
        2.  备选方案使用 `Readability.js` 进行语义化提取。
        3.  保底方案提取 `article` 或 `main` 标签内容。
    *   **阅读器遮罩**：在当前页生成全屏阅读 UI 覆盖层。
    *   **交互**：监听文本选择，支持“划词笔记”。

### 2.3 侧边栏 UI (`sidepanel/`)
*   **职责**：用户交互中心。
*   **功能模块**：
    *   **Notes (笔记)**：基于当前 URL 的笔记管理。支持标题、内容、引用（Quote）及搜索。存储于 `chrome.storage.local`。
    *   **Site Config (站点配置)**：管理自定义 CSS 选择器和排除列表。支持“自动探测（Detect Selector）”。
    *   **Settings (设置)**：配置自动弹出规则、阅读器外观（字体、字号）、Firebase 云端同步参数。
    *   **Account (账户)**：Google 登录集成。

### 2.4 专用阅读页面 (`readability/reader.html`)
*   **职责**：沉浸式阅读与高级导出。
*   **功能**：
    *   **个性化定制**：支持深色、浅色、护眼（Sepia）主题；动态调整字号、行高、内容宽度。
    *   **智能排版**：自动生成文章目录（TOC），支持滚动定位。
    *   **EPUB 导出**：集成 `epub-generator.js`，支持将文章打包为 EPUB 格式，并支持直接推送到电子书设备（如小米多看）。

## 3. 关键技术特性

### 3.1 内容提取流 (Content Extraction Flow)
扩展采用三级降级机制平衡准确性与通用性：
1.  **Selector Based**: 针对特定站点（如雪球、知乎）使用硬编码或用户定义的精确 CSS 选择器。
2.  **Logic Based**: 使用 `Readability.js` 的复杂启发式算法识别正文。
3.  **Semantic Based**: 依赖 HTML5 语义化标签。

### 3.2 EPUB 生成与传输
*   **生成**：利用 `JSZip` 在浏览器端构建标准的 EPUB 容器。
*   **清理**：`EPUBGenerator` 会对 HTML 进行 XHTML 规范化清理，移除危险脚本，确保在电子书阅读器上的兼容性。
*   **传输 (Smart CORS Upload)**：为解决局域网设备上传的跨域问题，采用多种策略（本地代理、Background Relay、公网代理）确保文件能成功推送到 e-reader。

### 3.3 数据存储
*   **本地存储**：`chrome.storage.local` 存储大体积数据（如每页笔记、站点选择器缓存）。
*   **同步存储**：`chrome.storage.sync` 存储用户偏好、URL 匹配规则等轻量配置。
*   **云端集成**：预留 Firebase Realtime Database 接口，支持多端数据同步。

## 4. 技术栈
*   **核心**: JavaScript (ES6+), HTML5, CSS3.
*   **依赖库**:
    *   `Readability.js`: 内容提取。
    *   `JSZip`: EPUB 打包。
    *   `Tippy.js`: UI 提示。
    *   `Tiptap / ProseMirror`: 预研/集成中的富文本编辑。
*   **部署/打包**: Chrome Extension MV3.
